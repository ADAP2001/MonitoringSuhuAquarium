# MonitoringSuhuAquarium

Bahan dan Alat yang diperlukan:

- ESP32 Bluetooth
- Sensor Suhu DS18B20
- Resistor 4.7K ohm (untuk sensor suhu)
- LED Biru (untuk indikator Dingin)
- LED Kuning (untuk indikator Normal)
- LED Merah (untuk indikator Panas)
